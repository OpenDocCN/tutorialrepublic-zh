# HTML 字符实体

> 原文：<https://www.tutorialrepublic.com/html-reference/html-character-entities.php>

以下部分给出了字符实体引用的完整列表。

下表列出了 HTML 中的基本实体。

| 字符 | 实体名称 | 实体编号 | 描述 |
| `&` | `&amp;` | `&#38;` | `Ampersand` |
| `"` | `&quot;` | `&#34;` | `Double quote mark` |
| `<` | `&lt;` | `&#60;` | `Less than symbol` |
| `>` | `&gt;` | `&#62;` | `Greater than symbol` |
| `'` | `&apos;` | `&#39;` | `Apostrophe (***XHTML only***)` |
